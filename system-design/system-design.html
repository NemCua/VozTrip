4

Feature Flag

Toàn bộ endpoint đều được bảo vệ bởi feature flag. Nếu flag bị tắt, endpoint trả về 404 { message: "Tính năng này hiện không khả dụng." } thay vì thực thi handler.

Request → EndpointFilter (WithFeatureFlag) → kiểm tra FeaturesConfig
flag = false → 404 ngay lập tức
flag = true → chạy handler bình thường

FeaturesConfig được inject qua DI, đọc từ appsettings.json hoặc environment variables.

F11 — Quản lý POI

GET/api/seller/pois

Mảng POI thuộc seller đang đăng nhập, sắp xếp CreatedAt DESC.

POST/api/seller/pois
Body: { "poiName": "Nhà thờ Đức Bà", "latitude": 10.7797, "longitude": 106.6990, "zoneId": "...", "triggerRadius": 15.0, "isActive": true }
Response: { "poiId": "..." }
Plan limit: Nếu flag PlanLimit.Enabled VÀ seller đang ở plan "free":
Đếm POI hiện tại → nếu >= FreePlanMaxPois → 400 { code: "PLAN_LIMIT" }
PUT/api/seller/pois/{id}

Cập nhật POI. Chỉ cho phép với POI thuộc seller đang đăng nhập.

DELETE/api/seller/pois/{id}
Cascade xóa Localization, Media, Question, VisitLog. Không dọn Cloudinary — nhiệm vụ của Admin.

F12 — Nội dung đa ngôn ngữ (Localization)

GET/api/seller/pois/{id}/localizations

Lấy tất cả localization của POI.

PUT/api/seller/pois/{id}/localizations/{languageId}
Body: { "title": "...", "description": "...", "audioUrl": null, "audioDuration": null }
Nếu chưa có record → tạo mới. Nếu đã có → cập nhật. Luôn set IsAutoTranslated = false.
DELETE/api/seller/pois/{id}/localizations/{languageId}
Xóa audio trên Cloudinary (nếu có AudioPublicId), sau đó xóa record.

F13 — Upload Audio thuyết minh

POST/api/seller/pois/{id}/localizations/{languageId}/audio

Content-Type: multipart/form-data · Field: file (audio/*)

Response: { "audioUrl": "https://res.cloudinary.com/..." }
1. Kiểm tra ContentType phải bắt đầu bằng audio/
2. Nếu đã có AudioPublicId cũ VÀ flag DeleteOldOnReplace.Enabled → xóa file cũ
3. Upload lên voztrip/audio/{sellerId}
4. Upsert AudioUrl, AudioPublicId

F14 — Dịch tự động (Auto Translate)

POST/api/seller/pois/{id}/localizations/translate
Body: { "sourceLanguageId": "..." }
Response: { "message": "Đã dịch sang N ngôn ngữ", "translated": 3 }
Bỏ qua ngôn ngữ đã có bản thủ công nếu flag SkipExistingManual.Enabled.
Gọi LibreTranslateService.TranslateToManyAsync() song song. Set IsAutoTranslated = true.

F15 — Quản lý Media

GET/api/seller/pois/{id}/media

Danh sách media của POI.

POST/api/seller/pois/{id}/media/upload

Content-Type: multipart/form-data · Field: file (image/* hoặc video/*)

Response: { "mediaId": "...", "mediaUrl": "...", "mediaType": "image" }
video/* → folder voztrip/videos/{sellerId} (flag Video.Enabled)
image/* → folder voztrip/images/{sellerId} (flag Image.Enabled)
DELETE/api/seller/media/{mediaId}
Xóa file trên Cloudinary theo PublicId + MediaType, sau đó xóa record.

F16 — Quản lý Q&A

POST/api/seller/pois/{id}/questions

Luồng nghiệp vụ chính (Business Flow)

Seller Onboarding (đăng ký → duyệt → tạo nội dung)

[Seller] Đăng ký → Trạng thái: ApprovedAt = null
[Admin] Duyệt → Trạng thái: ApprovedAt = now, ApprovedBy = adminId
[Seller] Đăng nhập → Nhận JWT token (7 ngày)
[Seller] Tạo POI → Kiểm tra plan limit (Free: max N POI)
[Seller] Nhập text → PoiLocalization (title + description)
[Seller] Dịch tự động → LibreTranslate → PoiLocalization cho các ngôn ngữ khác
[Seller] Upload audio → Cloudinary → cập nhật AudioUrl, AudioPublicId
[Seller] Upload media → Cloudinary → PoiMedia record
[Seller] Tạo Q&A → Question + Answer

Guest trải nghiệm

[App] Mở lần đầu → GET /api/languages → chọn ngôn ngữ
[App] Đồng ý policy → POST /api/sessions → tạo GuestSession
[App] Browse POI → GET /api/pois?languageId=...
[App] Xem chi tiết → GET /api/pois/{id}?languageId=...
[App] Nghe audio → Phát trực tiếp từ Cloudinary URL
[App] Nếu no audio → TTS đọc description (client-side)
[App] Xem Q&A → GET /api/pois/{id}/questions?languageId=...
[App] GPS Trigger → khoảng cách < TriggerRadius → POST /api/visitlogs

Ràng buộc & Quy tắc nghiệp vụ

#Quy tắcNơi thực thi
R01Username phải uniqueAuthRoutes.register: AnyAsync(Username)
R02Seller phải được Admin duyệt mới đăng nhập đượcAuthRoutes.login: ApprovedAt != null
R03Không thể khóa / xóa tài khoản adminAdminRoutes.toggle, .delete
R04Seller chỉ thao tác với POI của chính mìnhSellerRoutes: filter SellerId = sellerId từ JWT
R05Plan Free bị giới hạn số POISellerRoutes.createPoi: COUNT >= FreePlanMaxPois
R06Mỗi POI localization chỉ giữ 1 file audioSellerRoutes.uploadAudio: xóa AudioPublicId cũ trước
R07Auto-translate: IsAutoTranslated = true; thủ công: falseSellerRoutes.upsertLocalization, .translate
R08Seller VIP không bị giới hạn số POIpoiLimit = null nếu plan = "vip"
R09GPS trigger ghi log bất kể đã ghé thăm trước đó chưaGuestRoutes.visitlogs: luôn INSERT mới
R10Xóa POI từ Admin dọn cả Cloudinary (media + audio)AdminRoutes.deletePoi với CleanCloudinary flag
R11Seller xóa POI không dọn CloudinarySellerRoutes.deletePoi: chỉ xóa DB
R12Token JWT có hiệu lực 7 ngàyAuthRoutes.GenerateToken: AddDays(7)
