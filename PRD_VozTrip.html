Expo React Native

02 — thiet ke database

Cac bang da duoc toi gian hoa cho pham vi do an: bo VisitLogs.DistanceAtTrigger, GuestSessions.DeviceInfo, GuestSessions.LastSeenAt. Gop PoiContents + PoiAudios thanh PoiLocalizations. Tat ca ID dung TEXT (UUID) thay vi so tu tang. Database: PostgreSQL tren Neon. Media: Cloudinary (luu ca URL va PublicId de xoa duoc).
-- Quan tri nguoi dung CREATE TABLE users ( user_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, username VARCHAR(100) UNIQUE NOT NULL, password_hash VARCHAR(255) NOT NULL, role VARCHAR(20) NOT NULL, -- 'admin' | 'seller' full_name VARCHAR(200), email VARCHAR(200), is_active BOOLEAN DEFAULT TRUE, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Thong tin rieng cua seller CREATE TABLE sellers ( seller_id TEXT PRIMARY KEY REFERENCES users(user_id), shop_name VARCHAR(200) NOT NULL, shop_logo VARCHAR(500), contact_phone VARCHAR(20), description TEXT, approved_at TIMESTAMPTZ, approved_by TEXT REFERENCES users(user_id) ); -- Khu vuc / Khu trung bay CREATE TABLE zones ( zone_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, zone_name VARCHAR(100) NOT NULL, description TEXT ); -- Diem tham quan / Gian hang CREATE TABLE pois ( poi_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, seller_id TEXT NOT NULL REFERENCES sellers(seller_id), zone_id TEXT REFERENCES zones(zone_id), poi_name VARCHAR(200) NOT NULL, latitude DOUBLE PRECISION NOT NULL, longitude DOUBLE PRECISION NOT NULL, trigger_radius DOUBLE PRECISION DEFAULT 10.0, -- met is_active BOOLEAN DEFAULT TRUE, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Ngon ngu ho tro CREATE TABLE languages ( language_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, language_code VARCHAR(10) UNIQUE NOT NULL, -- 'en', 'zh', 'ko', 'ja' language_name VARCHAR(100), is_active BOOLEAN DEFAULT TRUE ); -- Noi dung + audio theo ngon ngu (gop PoiContents va PoiAudios) CREATE TABLE poi_localizations ( localization_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, poi_id TEXT NOT NULL REFERENCES pois(poi_id) ON DELETE CASCADE, language_id TEXT NOT NULL REFERENCES languages(language_id), title VARCHAR(200), description TEXT, audio_url VARCHAR(500), -- Cloudinary URL audio_public_id VARCHAR(200), -- Cloudinary public_id de xoa audio_duration INTEGER, -- giay UNIQUE (poi_id, language_id) ); -- Anh va video cua POI (luu ca public_id cua Cloudinary de xoa duoc) CREATE TABLE poi_media ( media_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, poi_id TEXT NOT NULL REFERENCES pois(poi_id) ON DELETE CASCADE, media_type VARCHAR(20) NOT NULL, -- 'image' | 'video' media_url VARCHAR(500) NOT NULL, -- Cloudinary URL public_id VARCHAR(200) NOT NULL, -- Cloudinary public_id de xoa sort_order INTEGER DEFAULT 0 ); -- Cau hoi thuong gap CREATE TABLE questions ( question_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, poi_id TEXT NOT NULL REFERENCES pois(poi_id) ON DELETE CASCADE, language_id TEXT NOT NULL REFERENCES languages(language_id), question_text VARCHAR(500) NOT NULL, sort_order INTEGER DEFAULT 0, is_active BOOLEAN DEFAULT TRUE ); -- Cau tra loi (giu them poi_id de query nhanh) CREATE TABLE answers ( answer_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, question_id TEXT NOT NULL REFERENCES questions(question_id) ON DELETE CASCADE, poi_id TEXT NOT NULL REFERENCES pois(poi_id), language_id TEXT NOT NULL REFERENCES languages(language_id), answer_text TEXT NOT NULL, audio_url VARCHAR(500), -- Cloudinary URL audio_public_id VARCHAR(200) -- Cloudinary public_id de xoa ); -- Phien khach (khong can dang nhap) CREATE TABLE guest_sessions ( session_id TEXT PRIMARY KEY, -- UUID tu app sinh ra language_id TEXT REFERENCES languages(language_id), started_at TIMESTAMPTZ DEFAULT NOW() ); -- Lich su tham quan CREATE TABLE visit_logs ( log_id TEXT PRIMARY KEY DEFAULT gen_random_uuid()::TEXT, session_id TEXT REFERENCES guest_sessions(session_id), poi_id TEXT REFERENCES pois(poi_id), triggered_at TIMESTAMPTZ DEFAULT NOW() );

03 — thu vien su dung

Expo React Native — app du khach

react-native-maps Hien thi ban do, gim POI, vong tron TriggerRadius. Dung Google Maps tren Android.
expo-location Lay toa do GPS lien tuc, tinh khoang cach den POI bang cong thuc Haversine.
expo-av Phat file audio mp3/m4a tu Cloudinary URL khi co san (seller da upload). Hien thi progress bar va thoi luong.
expo-speech Text-to-Speech fallback: neu POI chua co audio file, app tu doc noi dung bang giong may theo ngon ngu khach dang chon. Ket hop voi LibreTranslate: seller chi can viet tieng Viet, khach nuoc ngoai van nghe duoc bang ngon ngu cua ho. Chien luoc Hybrid: uu tien AudioUrl neu co, fallback sang TTS neu khong.
i18next + react-i18next Dich giao dien app (nhan, tieu de, thong bao) theo ngon ngu khach chon.
@react-native-async-storage Luu SessionId va LanguageId xuong local, giu nguyen khi tat app.
axios Goi REST API toi .NET backend.

Next.js — web seller & admin

next-auth Xu ly dang nhap, quan ly session, phan quyen admin / seller.
axios / fetch Goi API .NET tu server component va client component.
react-hook-form Xu ly form tao/sua POI, noi dung, nguoi dung.
@tanstack/react-query Cache va fetch du lieu tu API, tu dong refresh khi co thay doi.

.NET Minimal API — backend

Npgsql.EntityFrameworkCore.PostgreSQL ORM chinh cho PostgreSQL tren Neon, code-first, tu dong tao bang bang migration.
Microsoft.EntityFrameworkCore.Tools CLI de chay migration: dotnet ef migrations add, dotnet ef database update.
Microsoft.AspNetCore.Authentication.JwtBearer Phan quyen API bang JWT token, phan biet admin / seller / guest.
BCrypt.Net-Next Ma hoa password truoc khi luu vao password_hash.
CloudinaryDotNet Upload anh/audio len Cloudinary, nhan ve URL va public_id de luu vao DB. Dung public_id khi can xoa file.
Swashbuckle.AspNetCore Swagger UI de test API trong qua trinh phat trien.
LibreTranslate (self-hosted / API) Dich tu dong noi dung PoiLocalizations sang cac ngon ngu chua co. Backend goi HTTP den LibreTranslate, luu ket qua va danh dau is_auto_translated = true de phan biet voi noi dung seller tu viet.

04 — luong chuc nang du khach

guest — expo react native
  • Mo app lan dau
    App tu sinh GUID lam SessionId, luu vao AsyncStorage. Khong can dang nhap.
    GuestSessions.SessionId GuestSessions.StartedAt
  • Lan dau mo app — chon ngon ngu (LanguagePickerScreen)
    App kiem tra AsyncStorage xem da co languageCode chua. Neu chua co, hien LanguagePickerScreen voi 5 ngon ngu tinh (vi/en/zh/ko/ja) — khong phu thuoc API de tranh loi khi chua ket noi. Khach chon, luu languageCode + tao SessionId (UUID tu sinh) vao AsyncStorage, POST /api/sessions, chuyen sang HomeScreen. Nhung lan sau mo lai, bo qua buoc nay.
    Languages (static fallback) AsyncStorage: languageCode, sessionId POST /api/sessions
  • Xem danh sach POI — HomeScreen
    GET /api/pois tra ve danh sach POI IsActive=true. Hien thi dang list doc, moi card co: ten POI, shop name, trigger radius, chip GPS trigger. Co o tim kiem loc client-side. Moi card co nut play nho de nghe preview audio tai cho (khong can phai den vi tri thuc te) — dung Hybrid Audio: neu co AudioUrl thi phat bang expo-av, neu khong co thi TTS doc PoiName bang expo-speech.
    GET /api/pois PoiLocalizations.AudioUrl (neu co) POIs.PoiName (TTS fallback)
  • GPS trigger — banner thong bao + tu dong phat audio
    useGPS hook chay ngam voi expo-location (watchPositionAsync, accuracy=High, distanceInterval=5m). Moi cap nhat vi tri tinh khoang cach Haversine den tung POI. Neu distance < TriggerRadius va POI chua duoc trigger trong session nay: hien banner noi phia duoi man hinh ("Ban dang o gan [ten POI]"), tu dong phat audio theo Hybrid strategy, POST /api/visitlogs. Banner tu an sau 8 giay. Bam nut "Xem" tren banner de chuyen sang POIDetailScreen. Moi POI chi trigger 1 lan / session (luu Set trong bo nho).
    POIs.TriggerRadius PoiLocalizations.AudioUrl PoiLocalizations.LanguageId POST /api/visitlogs
  • Xem chi tiet POI — POIDetailScreen
    GET /api/pois/{id}?languageId=... tra ve media + localizations. Media: neu chi co 1 anh thi hien fullwidth hero; neu nhieu anh/video thi hien thumbnail strip ngang 64x64 ngay duoi hero, tap de doi anh chinh, badge "1/n" goc duoi phai. Language selector: pills VI/EN/ZH/KO/JA, ngon ngu nao chua co noi dung hien icon mic nho (danh dau se dung TTS). Audio player hybrid: nut play/pause, waveform decorative, badge phan biet "Audio goc" vs "Text-to-Speech". Section Gioi thieu hien description theo ngon ngu. POST /api/visitlogs khi vao man hinh.
    PoiMedia.MediaUrl / MediaType / SortOrder PoiLocalizations.Title / Description / AudioUrl POST /api/visitlogs
  • Hoi & Dap (Q&A accordion)
    GET /api/pois/{id}/questions?languageId=... Hien danh sach cau hoi dang accordion. Bam cau hoi de mo cau tra loi. Moi cau tra loi co nut nghe: neu Answer.AudioUrl co -> expo-av; neu khong -> expo-speech doc AnswerText (TTS). Badge phan biet "Nghe cau tra loi" vs "Doc (TTS)".
    Questions.QuestionText / IsActive Answers.AnswerText / AudioUrl
  • Doi ngon ngu
    Bam icon language tren HomeScreen header de quay lai LanguagePickerScreen. Cap nhat languageCode trong AsyncStorage va POST /api/sessions. Toan bo noi dung doi theo lap tuc khi quay lai HomeScreen.
    Languages AsyncStorage: languageCode POST /api/sessions

05 — chuc nang seller

seller — next.js dashboard
Seller chi thao tac duoc tren du lieu cua chinh minh. Moi query o backend phai filter SellerId = currentUser.SellerId.
Account / Profile
READ UPDATE
Xem va sua thong tin shop: ShopName, ShopLogo, ContactPhone, Description. Khong tu xoa duoc tai khoan.
POIs
CREATE READ UPDATE DELETE
Tao moi gian hang voi ten, toa do GPS, ban kinh trigger, khu vuc. Bat/tat IsActive de hien/an tren app.
PoiLocalizations
CREATE READ UPDATE DELETE
Them noi dung text gioi thieu va upload file audio .mp3 cho tung ngon ngu. Moi POI co the co nhieu ngon ngu.
PoiMedia
CREATE READ DELETE
Upload anh va video cho gian hang. Sap xep thu tu hien thi bang SortOrder.
Questions
CREATE READ UPDATE DELETE
Them cau hoi thuong gap cho tung ngon ngu. Bat/tat IsActive, sap xep SortOrder.
Answers
CREATE READ UPDATE DELETE
Viet noi dung tra loi va upload audio cho tung cau hoi. Moi cau hoi co dung 1 cau tra loi.

06 — chuc nang admin

admin — next.js dashboard
Admin thay duoc toan bo du lieu, khong bi gioi han boi SellerId. Co them quyen duyet seller va khoa tai khoan.
Users
READ UPDATE DELETE
Xem danh sach, sua thong tin, bat/tat is_active de khoa tai khoan. Viec tao user duoc xu ly qua /register hoac tao seller truc tiep.
Sellers — tao & duyet
CREATE READ UPDATE
Admin tu tao seller moi (POST /admin/sellers), hoac duyet seller tu dang ky (PUT /admin/sellers/{id}/approve). Seller chua duoc duyet thi khong dang nhap duoc.
Zones
CREATE READ UPDATE DELETE
Quan ly cac khu vuc trung bay. Seller se chon Zone khi tao POI.
Languages
CREATE READ UPDATE
Them ngon ngu moi, bat/tat IsActive. Ngon ngu IsActive = 0 bi an khoi app va dashboard.
POIs — toan bo
READ UPDATE DELETE
Xem tat ca POI cua moi seller. Co the an/xoa POI vi pham. Khong tao thay seller.
VisitLogs — thong ke
READ
Xem bao cao: POI nao duoc ghe nhieu nhat, gio cao diem, so luot truy cap theo ngay.
