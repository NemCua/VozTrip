voztrip — tourism guide app

// project specification — full stack overview · updated 2025

01 — tong quan he thong

Mobile — Du khach
Expo React Native

App cai tren dien thoai. Khong can dang nhap. GPS tu dong phat audio khi den gan POI. Quet QR code de xem nhanh chi tiet POI. 5-tab navigation.

Web — Seller & Admin
Next.js

Dashboard quan ly gian hang, noi dung, nguoi dung. Seller tao QR code cho tung POI, chon toa do bang ban do tuong tac, nang cap VIP de tao nhieu POI.

Backend — API
.NET Minimal API

REST API trung tam. Xu ly logic, phan quyen JWT, kiem tra gioi han plan, giao tiep PostgreSQL qua EF Core.

Database
PostgreSQL + EF Core

PostgreSQL tren Neon (cloud). EF Core code-first migration. Media luu tren Cloudinary, chi luu URL + public_id vao DB.

03 — thu vien su dung

Expo React Native — app du khach

expo-location Lay toa do GPS lien tuc, tinh khoang cach Haversine den tung POI. watchPositionAsync accuracy=High, distanceInterval=5m.
expo-av Phat file audio mp3/m4a tu Cloudinary URL. Hien thi progress bar va thoi luong.
expo-speech Text-to-Speech fallback khi POI chua co audio. Chien luoc Hybrid: uu tien AudioUrl neu co, fallback TTS theo ngon ngu khach.
expo-camera Quet QR code tren ScanScreen (tab 4). Su dung CameraView voi barcodeScannerSettings: [{barcodeTypes: ["qr"]}]. Parse format voztrip:poi:{poiId}, Vibration.vibrate(80) khi quet thanh cong, chuyen sang POIDetailScreen.
react-native-maps Hien thi MapScreen voi custom gold marker. Bottom sheet Animated.View khi tap marker (tuong thich Expo Go). Hien vi tri user lam blue dot.
i18n custom (src/i18n/translations.ts) File dich tinh noi bo, ham tr(key, languageCode). Ho tro 5 ngon ngu: vi/en/zh/ko/ja. Bao gom tab_scan: {vi:"Quet QR", en:"Scan", zh:"扫码", ko:"QR스캔", ja:"QR"}.
@react-native-async-storage Luu SessionId va LanguageId xuong local.
axios Goi REST API toi .NET backend. Base URL cau hinh qua EXPO_PUBLIC_API_URL trong .env.

Next.js — web seller & admin

next-auth Xu ly dang nhap, quan ly session JWT, phan quyen admin / seller.
@tanstack/react-query Cache va fetch du lieu tu API. Dashboard dung refetchInterval: 30000 de lam moi thong ke dinh ky.
react-qr-code Tao SVG QR code cho tung POI voi format voztrip:poi:{poiId}. Canvas API download PNG co label ten POI.
leaflet + OpenStreetMap Ban do tuong tac chon toa do GPS khi tao/sua POI. Khong can API key. Import dong qua next/dynamic({ssr:false}) de tranh loi SSR. Drag marker hoac click map de cap nhat lat/lng.
axios / fetch Goi API .NET tu client components.

.NET Minimal API — backend

Npgsql.EntityFrameworkCore.PostgreSQL ORM chinh cho PostgreSQL tren Neon, code-first, migration.
Microsoft.AspNetCore.Authentication.JwtBearer Phan quyen API bang JWT token, phan biet admin / seller / guest.
BCrypt.Net-Next Ma hoa password truoc khi luu.
CloudinaryDotNet Upload anh/audio, nhan URL + public_id. Dung public_id khi xoa file.
Swashbuckle.AspNetCore Swagger UI de test API trong qua trinh phat trien.
LibreTranslate (self-hosted / API) Dich tu dong noi dung PoiLocalizations sang cac ngon ngu chua co. Backend goi HTTP, luu ket qua va danh dau is_auto_translated.

04 — use cases

4 nhom actor chinh: Guest (du khach dung app), Seller Free (gioi han 1 POI), Seller VIP (khong gioi han), Admin. Cac use case duoi day mo ta luong chinh da duoc xay dung.

Guest — du khach

UC-G01
Chon ngon ngu
Guest
Mo app lan dau, chon 1 trong 5 ngon ngu (vi/en/zh/ko/ja). App luu languageId vao AsyncStorage. Moi lan sau mo app, toan bo noi dung hien thi theo ngon ngu da chon.
UC-G02
GPS tu dong trigger audio
Guest
Khach di chuyen den gan POI (khoang cach < TriggerRadius). App tu dong hien banner va phat audio thuyet minh (uu tien file Cloudinary, fallback TTS). Ghi VisitLog. Moi POI chi trigger 1 lan/session.
UC-G03
Quet QR code
Guest
Khach mo tab "Quet QR" (tab 4), quet QR code in tren bien hieu POI. App parse voztrip:poi:{id}, rung nhe, chuyen thang vao POIDetailScreen cua diem do.
UC-G04
Xem chi tiet POI
Guest
Xem anh, nghe audio, doc mo ta, xem Q&A. Doi ngon ngu truc tiep trong man hinh chi tiet. Neu ngon ngu chua co audio thi dung TTS. Vao tu HomeScreen, NearbyScreen, MapScreen hoac quet QR.
UC-G05
Xem ban do POI
Guest
Xem vi tri cac POI tren ban do, tap marker hien thong tin tom tat + nut chi tiet. Blue dot hien vi tri hien tai cua khach.

Seller — quan ly gian hang

UC-S01
Dang ky & dang nhap
Seller
Seller dang ky tai khoan, admin duyet. Sau khi duoc duyet (approved_at != NULL), seller dang nhap bang username/password, nhan JWT token.
UC-S02
Tao POI voi ban do
Seller FreeSeller VIP
Dien form tao POI: ten, toa do (nhap tay hoac chon tren ban do Leaflet/OpenStreetMap), ban kinh trigger. Seller Free bi gioi han 1 POI; vuot gioi han tra ve {code:"PLAN_LIMIT"}, chuyen trang upgrade.
UC-S03
Quan ly noi dung da ngon ngu
Seller
Them text mo ta va upload audio .mp3 cho tung ngon ngu (vi/en/zh/ko/ja). Them Q&A theo ngon ngu. Upload anh/video cho gallery.
UC-S04
Tao & tai QR code
Seller
Tao QR code dung react-qr-code (SVG, format voztrip:poi:{id}). Tai ve file PNG (Canvas API) co label ten POI. In dan len bien hieu gian hang.
UC-S05
Nang cap VIP
Seller Free
Seller nhap thong tin the (16 chu so), thanh toan 299.000d mot lan. Backend cap nhat plan='vip', plan_upgraded_at=NOW(). Sau do co the tao khong gioi han POI.
UC-S06
Xem thong ke gian hang
Seller
Xem dashboard: tong POI, luot tham quan hom nay, bieu do 7 ngay, top 5 POI duoc ghe nhieu nhat. Du lieu lam moi moi 30 giay.

Admin — quan tri vien

UC-A01
Duyet seller
Admin
Xem danh sach seller cho duyet, bam Approve de set approved_at. Seller chua duoc duyet khong the dang nhap.
UC-A02
Quan ly tat ca POI
Admin
Xem toan bo POI cua moi seller, loc theo ten/seller/trang thai. Toggle IsActive de an/hien POI vi pham khoi app guest.
UC-A03
Xem bao cao tong hop
Admin
Dashboard: tong seller, seller cho duyet, tong POI, POI dang hoat dong, tong luot tham quan, luot hom nay. Bieu do 7 ngay, top 5 POI toan he thong.

07 — luong chuc nang du khach

guest — expo react native (5-tab navigation)
Navigation: Bottom tab bar 5 tab — Kham pha | Gan day | Ban do | Quet QR | Cai dat. POIDetailScreen la full-screen overlay (khong co tab bar). GPS trigger chay ngam tren tat ca cac tab. Toan bo label UI dich qua ham tr(key, languageCode) trong src/i18n/translations.ts.
  • Mo app lan dau — LanguagePickerScreen
    App kiem tra AsyncStorage xem da co languageId chua. Neu chua, hien LanguagePickerScreen voi 5 ngon ngu (vi/en/zh/ko/ja). Chon langCode → tim languageId → luu AsyncStorage, tao SessionId, POST /api/sessions → chuyen HomeScreen.
    Languages (static list) AsyncStorage: languageId, sessionId POST /api/sessions
  • Tab Kham pha — HomeScreen
    GET /api/pois?languageId= tra ve POI IsActive=true kem thumbnailUrl va localizedName. List doc voi filter pills. GPS trigger chay ngam: khi vao vung trigger hien banner truot len, phat audio hybrid, POST /api/visitlogs.
    GET /api/pois?languageId= PoiLocalizations.Title POST /api/visitlogs
  • Tab Gan day — NearbyScreen
    Fetch cung GET /api/pois, tinh Haversine, sap xep tang dan. Badge xanh khi dang trong vung trigger. Nut play hybrid audio.
    GET /api/pois?languageId= expo-location: watchPositionAsync Haversine sort
  • Tab Ban do — MapScreen
    react-native-maps voi gold custom marker. Tap marker: Animated.View slide-up panel hien: thumbnail, ten, khoang cach, mo ta ngan, nut nghe + nut chi tiet. FAB recenter. GPS trigger van chay ngam.
    GET /api/pois?languageId= GET /api/pois/{id}?languageId= react-native-maps
  • Tab Quet QR — ScanScreen
    CameraView voi barcodeScannerSettings: [{barcodeTypes:["qr"]}]. Overlay voi 4 goc vang lam khung can. Khi quet duoc QR hop le (voztrip:poi:{id}): Vibration.vibrate(80), navigate POIDetailScreen. Hien UI xin quyen camera neu chua cap. Hint pill phia duoi man hinh. Quet 1 lan roi reset sau 2s de quet tiep.
    expo-camera: CameraView GET /api/pois/{id}?languageId= format: voztrip:poi:{poiId}
  • Xem chi tiet POI — POIDetailScreen (full-screen)
    GET /api/pois/{id}?languageId= tra ve media + localizations. Hero image fullwidth; thumbnail strip ngang khi nhieu anh. Language pills VI/EN/ZH/KO/JA. Audio player card: play/pause, badge "Audio goc" vs "TTS". Section Gioi thieu. Q&A accordion voi audio tung cau.
    PoiMedia PoiLocalizations Questions + Answers
  • Tab Cai dat — ProfileScreen
    Hien ngon ngu hien tai, grid 5 ngon ngu. Nut doi ngon ngu thuyet minh → LanguagePickerScreen. Thong tin app.
    AsyncStorage: languageId Languages static

08 — chuc nang seller

seller — next.js dashboard
Seller chi thao tac duoc tren du lieu cua chinh minh. Backend filter SellerId = currentUser.SellerId. Seller Free gioi han 1 POI — tao vuot tra {code:"PLAN_LIMIT"} → redirect /seller/upgrade.
Dashboard / Stats
READ
GET /api/seller/stats: totalPois, activePois, totalVisits, visitsToday, visitsByDay (7 ngay), topPois (5 POI nhieu luot nhat). RefetchInterval 30s.
POIs
CREATE READ UPDATE DELETE
Tao POI: form ten, toa do (nhap tay + MapPicker Leaflet), ban kinh trigger. Free plan: toi da 1 POI. Bat/tat IsActive.
QR Code
READ
Hien SVG QR (react-qr-code) cho tung POI voi format voztrip:poi:{id}. Nut tai PNG bang Canvas API co label ten POI.
Map Picker
UPDATE
Leaflet + OpenStreetMap (ssr:false). Click map hoac keo marker de chon lat/lng. Dong bo hai chieu voi input so.
PoiLocalizations
CREATE READ UPDATE DELETE
Them noi dung text va upload audio .mp3 cho tung ngon ngu.
PoiMedia
CREATE READ DELETE
Upload anh va video, sap xep SortOrder.
Questions & Answers
CREATE READ UPDATE DELETE
Them Q&A theo ngon ngu, upload audio tra loi.
VIP Upgrade
CREATE
So sanh Free vs VIP (299.000d mot lan). Form the tin dung mock (bat ky so 16 chu so). POST /api/seller/upgrade → plan='vip'.
